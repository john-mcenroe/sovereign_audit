# Scripts

Utility and one-off scripts for the project.

| Script | Purpose |
|--------|---------|
| `install_ffmpeg.sh` | Install ffmpeg (e.g. for video work) |
| `diagnose_cursor_network.sh` | Diagnose Cursor/network issues |
| `edit_video.py` | Video editing helper |
| `IMPLEMENTATION_EXAMPLE.py` | Code examples referenced by docs |

Run from project root or adjust paths as needed.
